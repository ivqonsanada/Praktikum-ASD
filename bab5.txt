DLL adalah suatu data struktur yang terhubung yang terdiri dari sekumpulan gudang simpanan yang biasa kita sebut dengan nodes. tiap nodes terdiri dari 2 field yang biasa kita sebut link. link inilah yang mengacu kepada node sebelum dan setelahnya dari sebuah linked list.

DLL adalah double linked list. Double yang artinya dua dan linked list adalah suatu list yang terhubung satu sama lain. jadi double linked list bisa diibaratkan dengan seseorang dengan dua tangan. jika single linked list bisa kita ibaratkan dengan seseorang dengan satu tangan yang memegang orang dibelakangnya, double linked list adalah orang dengan dua tangan. dia memegang orang didepan dan dibelakangnya.

contoh implementasi double linked list

public class Node {
    int data;
    Node next;
    Node prev;

    public Node(int data){
        this.data = data;
    }

}

public class DoubleLinkedList {
    Node first;
    Node last;

    public DoubleLinkedList() {
        first = null;
        last = null;
    }
}

ini adalah contoh implementasi pembuatan class dan konstruktor sederhana dengan DoubleLinkedList. kita membuat 2 node awal dan akhir. serta Node yang kita buat memiliki hubungan yaitu next dan prev sebagai tangan yang kita sebutkan di paragraf 2. Hal ini tentunya jelas berbeda dengan single linked list
